# Life Tracker

Полнофункциональное TUI-приложение для продуктивности. Работает целиком в терминале, управляется только клавиатурой — без мыши.

## Модули

| # | Модуль       | Описание |
|---|--------------|----------|
| 1 | **TODO**     | Задачи с приоритетами (high/med/low), дедлайнами, фильтром, прогресс-баром |
| 2 | **GYM**      | Запись тренировок, группы мышц, спарклайн прогресса веса, статистика |
| 3 | **HABITS**   | Ежедневный трекер привычек, стрики, недельная сетка (Mon-Sun) |
| 4 | **POMODORO** | Таймер work/break, авто-переключение, длинные перерывы каждые 4 сессии |

Dashboard — стартовый экран со сводкой по всем модулям и цитатой дня.

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python life.py
```

Приложение откроется в alternate screen buffer (как vim/htop). При выходе терминал вернётся в нормальное состояние.

## Горячие клавиши

### Глобальные

| Клавиша       | Действие                               |
|---------------|----------------------------------------|
| `1` `2` `3` `4` | Перейти в TODO / GYM / HABITS / POMODORO |
| `B` / `Esc`   | Назад на Dashboard                     |
| `?`           | Показать/скрыть справку                |
| `Q`           | Выход                                  |

### Навигация по спискам

| Клавиша       | Действие           |
|---------------|--------------------|
| `↑` / `K`     | Курсор вверх       |
| `↓` / `J`     | Курсор вниз        |

### TODO

| Клавиша       | Действие                          |
|---------------|-----------------------------------|
| `A`           | Добавить задачу (форма)           |
| `D` / `Delete`| Удалить выбранную задачу          |
| `Space`       | Переключить выполнено / не выполнено |
| `F`           | Сменить фильтр (all → active → done) |

### GYM

| Клавиша       | Действие                          |
|---------------|-----------------------------------|
| `A`           | Добавить тренировку (форма)       |
| `D` / `Delete`| Удалить выбранную запись          |
| `S`           | Статистика по выбранному упражнению |

### HABITS

| Клавиша       | Действие                           |
|---------------|------------------------------------|
| `A`           | Добавить привычку                  |
| `D` / `Delete`| Удалить выбранную привычку         |
| `Space`       | Отметить / снять отметку за сегодня |

### POMODORO

| Клавиша       | Действие                          |
|---------------|-----------------------------------|
| `S` / `Space` | Старт / Пауза / Следующий этап   |
| `R`           | Сброс таймера                     |
| `+` / `-`     | Изменить длительность работы (±1 мин) |

### Внутри форм

| Клавиша       | Действие                          |
|---------------|-----------------------------------|
| `Tab` / `Enter` | Следующее поле                  |
| `Enter` (на последнем) | Отправить форму        |
| `Esc`         | Отменить                          |
| `←` / `→`    | Выбор значения (select-поля)     |
| `Backspace`   | Удалить символ                    |

## Хранение данных

Все данные хранятся в `~/.life_tracker.json`. Файл создаётся автоматически при первом запуске.

## Требования

- Python 3.10+
- `rich` — рендер интерфейса
- `readchar` — чтение клавиш без Enter
- Работает по SSH (Termius, PuTTY и т.д.)

## Стек

- **Rich** — панели, таблицы, цвета, Live-перерисовка без мерцания
- **readchar** — мгновенная реакция на одиночные клавиши
- **threading** — фоновый тик таймера Pomodoro
- **Alternate screen buffer** — чистый выход, терминал не засоряется
